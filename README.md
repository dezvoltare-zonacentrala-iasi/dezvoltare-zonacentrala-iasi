# Creșterea atractivității zonei centrale a Municipiului Iași din punct de vedere socio-economic, cultural și turistic

Acest proiect conține o pagină HTML de prezentare pentru comunicatul de presă aferent investiției implementate de Municipiul Iași, în parteneriat cu Consiliul Județean Iași și Arhiepiscopia Iașilor, în cadrul Programului Regional Nord-Est 2021-2027, Prioritatea 7 - O regiune mai atractivă.

**Pagina live:** https://dezvoltare-zonacentrala-iasi.github.io/dezvoltare-zonacentrala-iasi/
